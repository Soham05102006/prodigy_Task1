# Task 01 – Responsive Landing Page with Interactive Navigation

This project is created as part of my internship task. The objective of this task was to build a responsive landing page with an interactive navigation menu using HTML, CSS, and JavaScript.

## 📌 Features

- Responsive landing page design
- Fixed navigation bar visible on all sections
- Navigation menu changes style when scrolling
- Hover effects on menu items
- Modern UI layout with sections like:
  - Home
  - Services
  - Portfolio
  - Testimonials
  - Contact
- Call-to-action buttons
- Clean and structured code

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Task Objective

Create an interactive navigation menu that:
- Changes color or style when the user scrolls
- Changes style when hovering over menu items
- Remains fixed at the top of the page
- Works smoothly across different screen sizes
